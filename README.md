# Internal Requests Manager

Internal Requests Manager is a front-end application built with Vue 3 that simulates an internal system used by companies to manage and track internal requests such as access, equipment, purchases, and support.

The project focuses on clean structure, realistic workflows, and a professional UI commonly found in internal tools.
